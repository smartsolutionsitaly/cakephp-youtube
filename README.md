YouTube plugin for [CakePHP](https://github.com/cakephp/cakephp)

## Installation
You can install _cakephp-youtube_ into your project using [Composer](https://getcomposer.org).

``` bash
composer require smartsolutionsitaly/cakephp-youtube
```
